## Етап 1: Налаштування, Інфраструктура та Авторизація (Core MVP)

### 1. Налаштування проєкту
- Ініціалізація проєкту Django
- Налаштування бази даних PostgreSQL
- Створення базових Django-додатків: `users`, `shots`, `social`

### 2. Модель Користувача
- Розширення вбудованої моделі користувача Django (Custom User Model)
- Додавання полів `Bio` та `Соціальні посилання`
- **Модуль:** `Users`

### 3. Аутентифікація
- Налаштування **DRF Simple JWT** для реєстрації, логіну та управління токенами (JWT)
- Реалізація захисту ендпоінтів за допомогою `IsAuthenticated`
- **Модуль:** `Users`

### 4. Google OAuth
- Інтеграція з **Google OAuth** через `django-rest-auth` або `django-allauth`
- Спрощена реєстрація через Google акаунт
- **Модуль:** `Users`

### 5. Профіль Користувача (GET/PATCH)
- Створення серіалізаторів та ViewSet для відображення та редагування профілю
- Поля: аватар, Bio, соціальні посилання
- **Модуль:** `Users`

---

## Етап 2: Основний Функціонал (Shots та Feed)

### 1. Модель Shots (Роботи)
- Створення моделі `Shot` з полями: `Назва`, `Опис`, посилання на `Користувача`, `Дата публікації`
- **Модуль:** `Shots`

### 2. Теги
- Створення моделі `Tag`
- Реалізація зв'язку Many-to-Many між `Tag` та `Shot`
- **Модуль:** `Shots`

### 3. Зберігання файлів
- Інтеграція **`django-storages`** для роботи з **AWS S3 / Cloudinary**
- Налаштування завантаження зображень та прев'ю при публікації Shot
- **Модуль:** `Shots`

### 4. CRUD Shots API
- ViewSet для публікації (`POST`), редагування (`PATCH`), видалення (`DELETE`) та отримання детальної сторінки Shot
- **Модуль:** `Shots`

### 5. Глобальний Feed (Стрічка)
- Ендпоінт для отримання списку всіх робіт
- Реалізація **пагінації** (`LimitOffsetPagination`)
- **Модуль:** `Shots`

---

## Етап 3: Соціальні Взаємодії та Пошук (MVP Features)

### 1. Лайки
- Створення моделі `Like` (зв'язок Користувач — Shot)
- Ендпоінт для додавання/видалення лайку: `POST /api/shots/{id}/like/`
- **Модуль:** `Social`

### 2. Коментарі
- Створення моделі `Comment`
- Ендпоінти для отримання та додавання коментарів до Shot
- **Модуль:** `Social`

### 3. Підписки
- Створення моделі `Follow` (зв'язок Користувач — Користувач)
- Ендпоінти для підписки/відписки на дизайнера
- **Модуль:** `Social`

### 4. Збереження (Favorites)
- Створення моделі `Favorite` для збереження робіт в обране
- **Модуль:** `Social`

### 5. Пошук
- Інтеграція з **`django-filter`** та/або повнотекстовий пошук через PostgreSQL
- Пошук за тегами, авторами та ключовими словами
- **Модуль:** `Shots`
